Glomerulus 병변 스크린

후각 수용 뉴런(ORN) 전체를 seed로 두고 glomerulus를 하나씩 무력화했을 때, 모델상 influence 분포가 얼마나 교란되는지 측정. 두 개의 방법론적 함정을 걷어낸 뒤의 최종 결과.

  • 최종 순위는 엣지(시냅스) 수준 병변에서 나옴 — 뉴런을 죽이는 대신 시냅스를 자름.

ρ = 두 순위표가 얼마나 같은지 (1 = 완전히 같은 순서, 0 = 무관). 모든 기호의 정의식은 바로 아래 「읽는 법」에.

읽는 법

이 문서의 용어

r — influence 벡터
(αW − I) r = −s # 이걸 풀어서 r을 얻음 W[i,j] = 뉴런 j → i 시냅스 수 (unsigned, count_thresh=5) s = seed 벡터 (ORN이면 1, 아니면 0) α = λ_target / λ_max(W), λ_target = 0.99 −I = leak 항 동치 급수: r = Σk≥0 (αW)k s # 모든 경로 길이의 합
  • r_ctrl = 병변 없는 정상 상태, r_les = 병변 후.
  • α는 온전한 W에서 한 번만 구해 모든 조건에 고정 (함정 ① 참조).
D_set / D_mag — 병변 피해량
cohort = topN(r_ctrl | universe) # control 기준 상위 N개, 고정 D_set = 1 − |cohort ∩ topN(r_les)| / N # 상위권 "명단"이 바뀐 정도 D_mag_sum = 1 − ( Σi∈cohort r_les[i] ) / ( Σi∈cohort r_ctrl[i] ) # 영향력 가중 낙폭 D_mag_mean = (1/N) · Σi∈cohort ( 1 − r_les[i] / r_ctrl[i] ) # 뉴런 동등 낙폭 Importance = ( D_set + D_mag ) / 2 전부 [0, 1]. 0 = 무피해, 1 = 완파.
  • 코호트는 control 기준으로 고정 — 병변 후 순위 밖으로 밀려난 뉴런도 명단에 남김. top_N(r_les)로 잡으면 가장 크게 다친 뉴런이 빠져나가 낙폭을 과소평가함.
  • top_N(r_les)는 D_set에만 등장. D_mag의 두 합은 인덱스 집합이 둘 다 cohort.
  • 실제로는 D_mag가 혼자 순위를 결정함 — Importance ~ D_mag의 ρ = 0.997. D_set은 값이 너무 작아(최대 0.022) 평균에 묻힘. 그리고 D_mag_sum과 D_mag_mean은 ρ = 1.000으로 완전히 같은 순위를 냄 (§05 상관행렬).
ρ — Spearman 순위 상관
ρ = corr( rank(x), rank(y) ) # 값이 아니라 "등수"끼리의 Pearson 상관 동점이 없을 때: ρ = 1 − 6·Σ di² / ( n·(n² − 1) ) di = 항목 i의 두 순위표상 등수 차이 n = 항목 수 (여기서는 glomerulus 54개) ρ = +1 : 두 순위가 완전히 같음 ρ = 0 : 무관 ρ = −1 : 정반대
  • 값의 크기가 아니라 순서만 비교하므로, 단위가 달라도 쓸 수 있음.
자기 PN
자기PN(g) = { p ∈ PN대상(g) : cell_type(p).split('_')[0] == g } 예: cell_type = "DA1_lPN" → 소속 = DA1 cell_type = "M_lvPNm46" → 소속 = M (다중 glomerulus, 특정 소속 없음) 전체 862 슬롯 중 자기 PN = 171
  • neuron.csv는 "glomerulus g에 수상돌기를 둔 뉴런" 목록 — 소속 목록이 아님.
  • DL2d 아래 등재된 PN 31개의 실제 구성:
    • DL2d_adPN, DL2d_vPN → 9개 = 자기 PN
    • DL2v_adPN×4, VL2a_vPN×3, DP1l_adPN … → 남의 glomerulus PN
    • M_adPNm8, M_lvPNm46 … → 다중 glomerulus PN
  • 핵심: "g에 가지가 있다" ≠ "g 소속이다".
단조성 (monotonicity)
D(A ∪ B) ≥ max( D(A), D(B) ) # 반드시 참이어야 함 증명: W ≥ 0 (unsigned) r = Σk (αW)k s 열을 더 0으로 만듦 → 급수에서 항을 더 뺌 → r은 감소만 가능 → D는 증가만 가능 ∎
  • → 발견이 아니라 자기검증 장치. 조건을 뒤섞거나, 병변 집합을 잘못 만들거나, solve가 조용히 틀린 답을 내면 이 부등식이 깨짐.
  • 실제로 α를 고정하기 전에는 깨졌고, 그게 함정 ①을 잡아낸 단서였음.
01

모델과 병변 정의

정상 상태

  • 선형 동역학의 정상 상태를 풂: (αW − I) r = −s.
  • W[i,j] = 뉴런 j → i 시냅스 수 (unsigned, count_thresh=5).
  • α = λ_target / λ_max(W), λ_target = 0.99. −I는 leak 항.
  • Seed s = ORN 2,931개 (orn.txt 2,971개 중 W에 존재하는 것).

병변

  • 엣지 수준 (최종 채택): glomerulus g의 ORN이 내보내는 모든 시냅스 제거.
  • ORN은 cell_type = ORN_<glom>으로 정확히 하나의 glomerulus에만 속함.
  • → ORN→X 엣지 19,245개가 각각 정확히 하나의 glomerulus에 귀속. 두 glomerulus의 엣지 집합 중복은 구조적으로 0.
  • 다중-glomerulus PN은 죽지 않음. g로부터의 입력만 잃고, 다른 입력과 자기 출력은 전부 유지.
02

함정 ① — α를 재계산하면 실험이 무의미해짐

Silent failure
라이브러리 기본값 (틀림): α_lesion = 0.99 / λ_max( W_lesion ) # 병변된 W에서 매번 다시 계산 이 스크린 (맞음): α_fixed = 0.99 / λ_max( W_intact ) # 온전한 W에서 한 번만, 전 조건 고정
  • 병변을 가하면 λ_max가 떨어짐 → α_lesion이 커짐 → 살아남은 시냅스가 전부 증폭됨.
  • 그리고 그 증폭량이 병변의 중요도에 비례 — 중요한 걸 끊을수록 λ_max가 많이 떨어지고, 그만큼 α가 크게 보정됨.
  • → 측정하려는 효과를 거의 정확히 상쇄함.

D = 병변으로 상위권 influence가 떨어진 정도 (0 = 무피해, 1 = 완파). 정의식은 「읽는 법」 참조.

  • 측정값 D의 95.6%가 증발함 — 평균적으로 원래의 4.4%만 남음.
  • 그런데 더 나쁜 건, 남은 4.4%가 진짜 D와 상관이 없다는 것 (ρ = −0.06).
    • 크기만 줄어든 게 아니라 순서가 뒤죽박죽이 됨.
    • DL2d는 진짜로 가장 교란적인데(0.589) 거의 0으로 뭉개지고, 원래 별거 아니던 놈이 위로 올라옴.
    • → "효과가 작게 측정된다"가 아니라 "아무것도 측정하지 못한다".
  • 6개 glomerulus는 D가 음수로 나옴 — 병변이 influence를 증가시킴 (물리적으로 불가능).
  • 병변이 작을수록 더 심각해짐 — α 드리프트의 절대 크기(~1%)는 작지만, 재려는 D도 작아서 비율로는 전부.

→ 온전한 W에서 α를 한 번만 구해 모든 조건에 고정. 그러면 control과 lesion 사이에 달라지는 것은 silence된 열의 제거뿐.

  • 부수 효과 (강력한 검증 장치): α가 고정이면 unsigned 모드에서 r_lesion ≤ r_control이 모든 뉴런에서 정확히 성립.
    • W ≥ 0이고 r = Σ(αW)ᵏ s → 열 제거는 급수에서 항을 빼는 것뿐.
    • 실측 최대 비율 1.00000000. α가 움직이면 이 성질이 깨짐.
03

함정 ② — 뉴런을 죽이면 병변이 glomerulus에 귀속되지 않음

Attribution contamination
  • neuron.csv의 PN 대상 862슬롯 중, 그 glomerulus에 실제로 속하는 PN은 171개뿐.
  • 나머지는 다중-glomerulus PN(M_*)이거나 다른 glomerulus의 PN.
  • 12개 glomerulus는 자기 PN이 0개 (DM1, VA6, VA2, VA7l, VM6l/m/v …).

  • 결과: Spearman(importance, 대상 PN 총수) = +0.580 vs Spearman(importance, 자기 PN 수) = +0.117.
  • 즉 순위는 "그 glomerulus에 얼마나 많은 회로가 가지를 뻗고 있나"를 재고 있었지, "그 glomerulus 고유의 출력 경로가 얼마나 중요한가"를 재고 있지 않았음.
  • 엄격한 uniglomerular 변종은 만들 수 없음 — 12개가 빈 집합.

→ 뉴런이 아니라 시냅스를 자르면 문제가 사라짐. ORN 쪽은 귀속이 모호하지 않기 때문.

04

지표

D_set · D_mag · Importance의 정의식은 위 「읽는 법」 참조. 여기서는 그 식이 어느 뉴런 집합 위에서 계산되는지를 고정함.

Universe — 순위를 매기는 뉴런 집합. 모든 조건에서 동일해야 함

universe = 전체 뉴런 (150,299) − seed ORN (2,931) # s로 직접 구동됨 → 자명하게 상위권 − 어떤 조건에서든 silence되는 뉴런 (781) # 점수가 병변의 산물 − r_ctrl == 0 인 뉴런 # D_mag_mean의 0으로 나누기 방지 = 126,860
  • 조건마다 universe가 다르면 순위를 비교할 수 없음 → 한 번 정해 전부에 고정.

N (상위 몇 개를 볼 것인가)

  • N = 500이 헤드라인.
  • N 선택이 결과를 만들지 않음 — N을 바꿔가며 매긴 순위끼리의 ρ: 0.930(N=50) ~ 0.997(N=250·1000). 즉 어떤 N을 골라도 거의 같은 순위.

각 행 = glomerulus 하나의 ORN 시냅스를 전부 잘랐을 때의 결과. scope = orn, N = 500. ORN 수는 그 glomerulus의 수용 뉴런 개수, 엣지 수는 그 ORN들이 만드는 연결 개수, 시냅스 수는 그 연결들의 총 시냅스 개수(= 실제로 잘라낸 양). 막대는 최대값(DP1l = 0.1129) 기준 상대 길이. 상위 6개 강조.

이 순위를 읽는 법

  • 순위 ≈ 시냅스 개수 순위 — ρ(Importance, ORN 시냅스 수) = +0.849.
    • 즉 시뮬레이션을 안 돌리고 ORN 시냅스 개수만 세도 순위가 거의 나옴.
    • 이게 틀렸다는 뜻은 아님 — DP1l이 시냅스 13,195개를 갖는 건 진짜고, 끊으면 진짜로 더 망가짐.
    • 다만 "숨은 요충지를 발견했다"는 주장은 못 함. 결과가 말하는 건 "입력 채널이 큰 놈이 크게 망가진다".
  • 크기를 보정하지 않음. 엣지 수준에서 크기는 진짜 생물학이기 때문 — "어느 glomerulus를 끊으면 가장 많이 망가지나"가 질문이면, 답에 크기가 포함되는 게 맞음.
  • D_set이 거의 0 (최대 0.022) → 병변은 상위권 명단을 뒤섞기보다 전체를 아래로 끌어내림. Importance는 사실상 D_mag(점수 낙폭)가 지배.

→ "DP1l이 특별한 요충지다"가 아니라 "DP1l이 뇌로 가장 많은 후각 입력을 보낸다"로 읽을 것.

  • → 어떤 glomerulus를 죽이든 공유 LN 뭉치가 함께 날아가 54개 조건이 서로 닮아버림.

orn_local — ORN 엣지 + glomerulus 내부 엣지 (엣지 수준) 폐기

  • D_mag_sum: min 0.388 / 중앙 0.888 / max 0.943. 46/54가 D > 0.80.
  • glomerulus당 국소 엣지를 79~4,100개(중앙값 1,507) 자르는데, 그게 AL의 공유 내부 배선.
  • 시냅스 좌표가 없어 "둘 다 g에 가지를 뻗었으면 g 안의 시냅스" 휴리스틱을 썼고, 실패.
  • all 변종과 똑같은 실패 양상.

orn — ORN 엣지만 채택

  • D_mag_sum: 0.0004 – 0.204. 건강한 동적 범위.
  • N에 강건 (ρ = 0.930 ~ 0.997). 귀속 모호성 0.

검증

  • 아래는 전부 발견이 아니라 자기검증 — 수학적으로 참이어야 하는 것들이고, 깨지면 코드 버그라는 뜻.
  • 빈 병변 = control — 정확히 일치 (bit-exact).
  • 단조성 — D(A∪B) ≥ max(D(A), D(B)): 1,431쌍 전부 위반 0건.
    • A와 B를 같이 끊으면 하나만 끊었을 때보다 반드시 더 망가져야 함 (열을 더 0으로 만드는 건 급수에서 항을 더 빼는 것뿐이므로).
    • α 고정 전에는 이게 깨졌고, 그게 함정 ①을 잡아낸 단서였음.
  • α 고정 검증 — max(r_lesion / r_control) over 전 뉴런 = 1.00000000 (병변이 influence를 늘릴 수 없어야 함).
  • 엣지 기계 교차검증 — orn scope의 엣지 제거가 열 0으로 만들기와 정확히 일치해야 함 (ORN의 출력 전부 = ORN의 열). 스크립트가 실제로 대조하고 불일치 시 즉시 중단. 통과.
  • 모든 지표 ∈ [0,1] — clip 전 원값 기준.
  • 전체 실행을 -W error::RuntimeWarning으로 수행 — 경고 하나라도 뜨면 중단. 무경고 완주.

한계

  • 결과는 모델상 influence 분포의 변화이지, 행동 예측이 아님.
  • ORN 88개는 ORN_<glom> 형식의 cell_type이 없어 glomerulus 매핑에서 제외.
  • seed ID 40개, 대상 ID 3개는 W에 없어 제외 (target_audit.csv에 기록).
  • glomerulus 내부(AL 국소 회로) 병변은 측정 불가 — 엣지 리스트에 시냅스 공간 좌표가 없음. orn_local 휴리스틱은 포화로 실패.
  • 순위는 시냅스 수와 강하게 연동(ρ=0.849). "크기 대비 특별한 곳"은 이 스크린이 답하지 않는 질문.
  • 조합 분석은 뉴런 수준(오염된 대상 정의)에서만 수행됨.
  • unsigned 모드 — 억제성 시냅스를 음수로 다루지 않음.
